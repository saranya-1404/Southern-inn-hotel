# Southern Inn – Luxury Boutique Hotel Website

This is a responsive frontend hotel website developed as part of the MANTRA 2026 Summer School Web Development assignment.

## Website Type

Luxury Boutique Hotel Website

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub
- Netlify

## Pages Included

- Home
- About
- Rooms
- Amenities
- Gallery
- Reviews
- Booking
- Contact

## Features

- Responsive design for desktop, tablet and mobile devices
- SEO-friendly page titles, meta descriptions and keywords
- Image alt text for accessibility
- Mobile navigation menu
- Booking enquiry form with JavaScript validation
- Luxury pastel-themed user interface
- Gallery showcasing hotel interiors and rooms
- Guest reviews section

## Hotel Highlights

- Jacuzzi King Room
- Jacuzzi Queen Room
- Sea View King Room
- Sea View Queen Room
- Double Bed Family Room
- Elegant Lobby
- Garden Views
- Luxury Amenities

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.

## Deployment

The project can be deployed using GitHub and Netlify for free hosting.
